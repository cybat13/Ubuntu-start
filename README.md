# Ubuntu Lite Gaming Setup

Lightweight Ubuntu setup script for gaming, dev tools, Vulkan, GameMode, MangoHud, Steam, Lutris, and laptop power profile toggles.

## Features

- Steam + Lutris + Wine + Winetricks
- GameMode + MangoHud
- Vulkan / Mesa userspace
- Microcode install
- NVIDIA auto install when detected
- Flatpak + Flathub
- Lightweight dev tools
- SSD trim
- Laptop power mode helpers

## Run

### Safer method: clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/ubuntu-lite-gaming.git
cd ubuntu-lite-gaming
chmod +x install.sh
./install.sh
